# Birthday Buddy 🎈

Never forget a birthday again. Smart reminders + gift suggestions with Amazon integration.

## 🚀 Features
* 🎂 Track friends & family birthdays
* 🎁 Smart gift suggestions (Affiliate integrated)
* 🔔 Custom reminders
* 💻 Cross-platform: Android, Windows, Mac, Linux
